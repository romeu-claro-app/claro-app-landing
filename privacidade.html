Política de Privacidade

Última atualização: maio de 2025

Resumo: Recolhemos apenas os dados necessários para prestar o serviço. Não vendemos os teus dados a terceiros. Tens o direito de aceder, corrigir e eliminar os teus dados a qualquer momento.

1. Responsável pelo Tratamento

O responsável pelo tratamento dos dados pessoais é o Claro-app.ch, contactável através de suporte@claro-app.ch.

2. Dados que Recolhemos

Recolhemos os seguintes dados pessoais:

  • Dados de registo: nome, apelido, endereço de email, número de telemóvel, data de nascimento, morada.
  • Dados de perfil: fotografia de perfil (opcional), cidade de residência, agregado familiar (opcional).
  • Dados de utilização: documentos submetidos para análise, histórico de análises, datas de acesso.
  • Dados de pagamento: processados pelo Stripe — não armazenamos dados de cartão de crédito.
  • Dados de contacto: email fornecido para receber notificações de lançamento.

3. Finalidade do Tratamento

Os teus dados são utilizados para:

  • Fornecer e melhorar os serviços do Claro-app.
  • Gerir a tua conta e subscrição.
  • Enviar comunicações de serviço (confirmações, lembretes de prazo).
  • Processar pagamentos de forma segura.
  • Cumprir obrigações legais.

4. Base Legal

O tratamento dos teus dados baseia-se no consentimento prestado no momento do registo, na execução do contrato de prestação de serviços, e em obrigações legais aplicáveis nos termos da lei suíça (nLPD) e do Regulamento Geral de Proteção de Dados (RGPD).

5. Partilha de Dados

Não vendemos nem partilhamos os teus dados pessoais com terceiros para fins comerciais. Partilhamos dados apenas com:

  • Supabase: armazenamento seguro de dados (servidores na Europa).
  • Stripe: processamento de pagamentos (certificado PCI DSS).
  • Anthropic (Claude AI): processamento dos documentos para análise — os documentos são enviados de forma segura e não são utilizados para treinar modelos.
  • Calendly: gestão de marcações de consulta.

6. Retenção de Dados

Os teus dados são conservados enquanto a tua conta estiver ativa. Após o encerramento da conta, os dados pessoais são eliminados no prazo de 30 dias, com exceção dos dados necessários para cumprimento de obrigações legais (máximo 10 anos).

7. Os Teus Direitos

Tens direito a:

  • Acesso: solicitar uma cópia dos teus dados pessoais.
  • Retificação: corrigir dados incorretos ou incompletos.
  • Eliminação: solicitar a eliminação dos teus dados ("direito ao esquecimento").
  • Portabilidade: receber os teus dados num formato estruturado.
  • Oposição: opor-te ao tratamento dos teus dados para fins de marketing.
  • Revogação do consentimento: a qualquer momento, sem prejuízo do tratamento anterior.

Para exercer estes direitos, contacta-nos através de suporte@claro-app.ch.

8. Segurança

Implementamos medidas técnicas e organizacionais adequadas para proteger os teus dados, incluindo encriptação em trânsito (HTTPS/TLS) e em repouso, autenticação segura, e controlo de acesso baseado em funções.

9. Cookies

O Claro-app utiliza apenas cookies técnicos essenciais para o funcionamento da sessão. Não utilizamos cookies de rastreamento ou publicidade.

10. Menores

O Claro-app não é destinado a menores de 18 anos. Não recolhemos intencionalmente dados de menores.

11. Alterações a esta Política

Podemos atualizar esta Política de tempos a tempos. As alterações serão comunicadas por email com antecedência mínima de 30 dias.

12. Contacto e Reclamações

Para questões de privacidade, contacta-nos em suporte@claro-app.ch. Tens também o direito de apresentar reclamação junto da autoridade de proteção de dados competente.
